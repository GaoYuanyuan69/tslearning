# TypeScript基础语法
## 变量声明
ts中变量声明如下所示：  
```
let 变量名: 变量类型 = 变量值;
```
例如：  
```typescript
let a: string = "Hello World!";
```
基础变量类型：
- any: 任意类型 
- number: 数值
- string: 字符串
- boolean: 布尔型
- {type}[]: 数组，例如：`let nums: number[] = [1, 2, 3];`
- [{type1}, {type2}, ... {typeN}]: 元组，例如：`let x: [number, string]; x = [1, "1"];`
- enum: 枚举
- void: 无返回值
- null: 空值
- undefined: 未指定
- never: 从不会出现的类型
